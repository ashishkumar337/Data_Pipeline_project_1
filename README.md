# Data Pipeline Project

## Overview
This project demonstrates a simple ETL (Extract, Transform, Load) pipeline using Python, Pandas, and Scikit-learn.

## Steps
1. Extract data from `data/raw_data.csv`.
2. Transform data (handle missing values, encode categorical features, scale numerical columns).
3. Load processed data into `output/processed_data.csv`.

## How to Run
```bash
pip install pandas scikit-learn
python pipeline.py
```

## Output
A clean and transformed CSV file saved in the `output` folder.
